## ⚠️ GitHub Actions Note

The CI/CD workflow file (`.github/workflows/python-tests.yml`) is included locally but couldn't be pushed due to OAuth token scope limitations.

**To enable GitHub Actions:**

Option 1 - Refresh token with workflow scope:
```bash
gh auth refresh -s workflow
git push origin main
```

Option 2 - Upload manually:
- Go to repository on GitHub
- Create `.github/workflows/` folder
- Upload `python-tests.yml` from your local copy
